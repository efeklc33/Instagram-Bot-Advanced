=== INSTAGRAM BOT KULLANIM KILAVUZU ===

1. Gereksinimler:
   - Windows işletim sistemi (Derlenen .exe için).
   - İnternet bağlantısı.
   - proxy.txt dosyası .exe ile aynı klasörde olmalıdır.

2. Kurulum:
   - InstagramBot.exe ve proxy.txt dosyalarını aynı klasöre koyun.

3. Kullanım:
   - InstagramBot.exe dosyasını çalıştırın.
   - Ana menüden yapmak istediğiniz işlemi seçin:
     1: Hesap Oluştur (Tarayıcı açılır, kayıt formunu doldurmanız gerekir).
     2: Takip Gönder (Giriş yaptıktan sonra hedef kullanıcı adını girin).
     3: Beğeni Gönder (Giriş yaptıktan sonra gönderi linkini girin).
     4: Çıkış.

4. Önemli Notlar:
   - Instagram'ın bot korumaları nedeniyle işlemler arasında beklemeler olabilir.
   - Proxylerin kalitesi botun başarısını doğrudan etkiler.
   - Hesap oluşturma aşamasında manuel müdahale (Captcha vb.) gerekebilir.
